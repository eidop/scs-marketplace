# Image & Video Manipulator Skill

This skill provides versatile capabilities for working with image and video media.

## Core Functionality

- **Image Editing:** Resize, crop, rotate, apply filters, adjust colors.
- **Video Processing:** Trim, merge, convert formats, extract frames.
- **Format Conversion:** Convert between various image (JPG, PNG, WebP) and video (MP4, GIF) formats.
- **Analysis:** Extract metadata, identify objects, generate descriptions.
- **Watermarking:** Apply custom watermarks to media.

## Usage Examples

- "Resize `image.jpg` to 800x600 pixels."
- "Convert `video.mp4` to a GIF."
- "Extract metadata from `photo.png`."
- "Trim `clip.mp4` from 0:10 to 0:30."
