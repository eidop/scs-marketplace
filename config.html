Templates

SCS provides pre-built .n8n and .json templates to accelerate your deployment. All templates are compatible with n8n version 1.0+ and SCS Memo-Bot.

Importing a Template

  1. Download the template from the SCS Store.
  2. Open your n8n instance and click Import from File.
  3. Configure the SCS_API_KEY in your environment variables.
{
  "name": "SCS Sales Outreach",
  "nodes": [...],
  "connections": {...},
  "settings": {
    "executionTimeout": 3600
  }
}

API Reference

Integrate SCS directly into your stack using our REST API. All requests must be authenticated using a Bearer token.

Endpoint: Send Message

POST https://api.scs.io/v1/memo-bot/message

curl -X POST https://api.scs.io/v1/memo-bot/message \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "How do I reset my password?",
    "context": "support",
    "user_id": "user_123"
  }'

Response

{
  "status": "success",
  "data": {
    "response": "To reset your password, visit the login page and click 'Forgot Password'.",
    "confidence": 0.98,
    "source": "kb_article_402"
  }
}

Environment Variables

Configure your SCS instance using these global variables.

Variable Description Default
SCS_API_KEY Your unique enterprise access key null
SCS_MODEL Preferred LLM (gpt-4o, claude-3-5) gpt-4o
SCS_LOG_LEVEL Verbosity of system logs info
SCS_MAX_TOKENS Limit per agent interaction 4096

Troubleshooting

Common issues and their resolutions.

Error: 401 Unauthorized

Your SCS_API_KEY is either missing or has expired. Verify the key in your SCS Console.

Error: 429 Rate Limit Exceeded

You have exceeded the request limit for your current plan. Consider upgrading to our Enterprise Plan for unlimited throughput.
